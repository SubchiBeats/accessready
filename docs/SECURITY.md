# Security and Privacy

AccessReady's browser demo scans uploaded files locally in the browser. The demo does not intentionally upload files to a server.

The CLI scans files locally on your machine or in your CI environment.

Do not upload sensitive, confidential, or controlled documents to hosted demos unless you control the deployment and have approval to process those files.

## Reporting security issues

Open a private security advisory in GitHub or contact the maintainers using the repository's preferred contact method.
