# Roadmap

## v0.1 Founder MVP

- Core scanner for HTML, Markdown, DOCX, PPTX, PDF preflight
- CLI
- Web upload interface
- Markdown/JSON reports
- Manual checklist
- GitHub Action workflow

## v0.2 Comms workflows

- CSV remediation tracker
- Better report branding
- Severity filters
- Project-level issue grouping
- Social content checks for alt text, hashtags, link text, and transcripts

## v0.3 Source-file improvement

- Improved DOCX/PPTX reading order heuristics
- Better table detection
- PowerPoint slide reading order warnings
- InDesign/Canva export checklist templates

## v0.4 AI-assisted but human-approved

- Alt text draft queue
- Chart-summary draft support
- Human approval status tracking
- Exportable alt text log

## v0.5 CI and team adoption

- Pull request comments
- GitHub Checks API integration
- Baseline mode for legacy content
- Suppression comments with justification
- Organization policy configuration

## v1.0 Professional workflow

- Evidence packet generator
- Client-ready PDF report
- VPAT/ACR support helper
- Optional external adapters for Acrobat/PAC/CommonLook outputs
